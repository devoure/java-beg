# :bank: Mortgage CLI
> :bulb: **Project** 3
## 💬 Description
> This is a simple java CLI app that calculates the mortgage from inputs of principal , period and annual rate.
## 🔧 Code Setup
   1. Clone the repo
        ```
            git clone https://github.com/devoure/java-beg.git 

        ```
   1. Move into the project file
        ```bash
            cd mortgage-cli

        ```
   1. Run the code
        ```bash
            javac Main.java
            java Main

        ```
