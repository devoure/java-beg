# :arrows_clockwise: Eulers
> :bulb: **Project** 4
## 💬 Description
> This is a simple java CLI app that gets eulers constant as the sum of a series of n values.
## 🔧 Code Setup
   1. Clone the repo
        ```
            git clone https://github.com/devoure/java-beg.git 

        ```
   1. Move into the project file
        ```bash
            cd eulers

        ```
   1. Run the code
        ```bash
            javac Main.java
            java Main

        ```
